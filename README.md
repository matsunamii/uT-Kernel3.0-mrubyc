# TRON × mruby/c リアルタイムスクリプト実行環境

## 概要

本プロジェクトは、NUCLEO-H533RE 上の μT-Kernel 3.0 BSP2 で mruby/c を動作させ、Ruby スクリプトを RTOS タスクとして並列実行するためのリアルタイムスクリプト実行環境です。

μT-Kernel 3.0 のタスク管理機能と、mruby/c の軽量 Ruby 実行環境を組み合わせることで、組込みシステム上で Ruby による制御ロジックを実行できる環境を構築しました。

動作確認用のサンプルコードとして `led.rb` と `print.rb` を用意しています。`led.rb` はボード上 LED を周期的に点滅させ、`print.rb` は LED の状態を読み取ってシリアル出力に表示します。これにより、複数の Ruby スクリプトを μT-Kernel 3.0 上の RTOS タスクとして並列実行できることを確認できます。

## 特徴

- μT-Kernel 3.0 BSP2 上で mruby/c を実行
- Ruby スクリプトを RTOS タスクとして並列実行
- STM32 HAL を用いた GPIO クラスを mruby/c に追加

## 使用環境

| 項目 | 内容 |
|---|---|
| マイコンボード | NUCLEO-H533RE |
| RTOS | μT-Kernel 3.0 BSP2 |
| IDE | STM32CubeIDE |
| 動作確認IDE | STM32CubeIDE 2.1.1 |
| 使用言語 | C, Ruby |
| Ruby実行環境 | mruby/c |
| 外部ハードウェア | なし |

## ドキュメント

本プロジェクトでは、実行手順、操作方法、開発者向けの拡張方法、動作確認方法を `docs/` ディレクトリにまとめています。
各ドキュメントの役割は以下の通りです。

### セットアップ・実行手順書

`setup_and_run_guide.md` では、本プロジェクトを実行するための一連の手順を説明しています。  
GitHubからプロジェクトをZIP形式でダウンロードし、STM32CubeIDEにインポートしてビルド・デバッグ実行し、シリアル出力を確認するまでの流れを記載しています。

### 開発者向けガイド

`developer_guide.md` では、本プロジェクトを拡張するための方法を説明しています。  
新しいRubyプログラムの追加方法、Rubyスクリプトをmruby/c用バイトコードとして組み込む方法、C言語側で実装した関数をRubyメソッドとして登録する方法、Rubyタスクを追加する方法などを記載しています。  
また、μT-Kernelのアラーム機能を用いたmruby/c VMのプリエンプション制御の仕組みについても説明しています。
